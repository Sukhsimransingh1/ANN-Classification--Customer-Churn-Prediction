# ANN-Classification--Customer-Churn-Prediction
This project is a Customer Churn Prediction system built using an Artificial Neural Network (ANN) for binary classification. It predicts whether a customer is likely to leave a bank based on their demographic and financial information. The trained model is deployed using a Streamlit web application for interactive use.
